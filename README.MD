# PROJETO PIZZARIA NANOSABORES

![alt text](/assets/LogoNanoSabores.png)

## Sobre

#### Este é um sistema de gerenciamento desenvolvido para a pizzaria fictícia NanoSabores

## Escopo

#### O projeto possui três telas principais: a tela do Cliente, do Administrador e da Cozinha

## Cliente

#### O cliete pode realizar pedidos através de um cardápio que interage diretamente com a cozinha

## Cozinha

#### Os pedidos são registrados e enviados para a cozinha em ordem, permitindo uma melhor visualização dos preparos pendentes

## Administrador

#### O administrador tem acesso a informações sobre funcionários, estoque, vendas e produtos, podendo emitir relatórios e cadastrar novas informações.
